# Mergington High School Security Policy

## Reporting a Vulnerability

At Mergington High, we take the security of our Extra-Curricular Activities website seriously.

1. Do not create a public issue about the vulnerability.
2. Go to the Security tab of this repository.
3. Click "Report a vulnerability".
4. Fill out the form with details about the issue.

## Response Timeline

- Report acknowledgment within 2 days
- Initial assessment within 5 days
- Critical issues fixed immediately

## Thank You

Responsible disclosure helps protect our entire school community.
